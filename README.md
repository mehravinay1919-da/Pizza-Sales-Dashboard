Overview
This project analyses pizza sales data using MySQL and Power BI to uncover business insights related to sales performance, customer ordering behaviour, revenue trends, and product performance.

The workflow includes:
- Loading and cleaning raw datasets
- Writing SQL queries for business analysis
- Creating KPIs and trend analysis
- Building an interactive Power BI dashboard
- Generating actionable business insights

The project demonstrates practical skills in:
- SQL
- Data Cleaning
- Data Analysis
- Business Intelligence
- Dashboard Development
- Data Visualization

Dataset
The dataset contains transactional pizza sales data with information related to:
- Orders
- Order Details
- Pizza Types
- Pizza Sizes
- Categories
- Prices
- Quantity Sold
- Order Date & Time

Main Tables
| Table Name      | Description                         |
| --------------- | ----------------------------------- |
| `orders`        | Stores order date and time          |
| `order_details` | Stores quantity and pizza ordered   |
| `pizzas`        | Contains pizza size and price       |
| `pizza_types`   | Contains pizza names and categories |

Tools & Technologies Used: 
- MySQL, 
- Power BI, 
- MS Excel  

Project Workflow
1. Data Loading
- Imported raw CSV datasets into MySQL database
- Created relational tables using primary and foreign keys
2. Data Cleaning
- Renamed tables for consistency
- Checked for missing and duplicate values
- Validated relationships between tables
- Standardized data formats
3. SQL Analysis
Performed business analysis using SQL queries such as:
- Total Orders
- Total Revenue
- Average Order Value
- Total Pizzas Sold
- Daily & Monthly Order Trends
- Best & Worst Selling Pizzas
- Revenue Contribution by Category
- Hourly Order Distribution
- Top Performing Pizza Types
- Cumulative Revenue Analysis
4. Power BI Dashboard
Built an interactive dashboard with:
- KPI Cards
- Revenue Analysis
- Top/Bottom Selling Pizzas
- Order Trends
- Quantity Analysis
- Dynamic Filters and Slicers
5. Reporting
Converted analysis into business insights for decision-making.


Key Insights:
- Classic Deluxe Pizza generated the highest number of orders.
- Thai Chicken Pizza contributed the highest revenue.
- Large-sized pizzas were the most commonly ordered.
- Peak orders were observed during lunch and evening hours.
-  Classic category contributed the highest overall sales volume.
